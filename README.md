本博客使用了Astro-paper的模板来创建，此服务部署在Cloudflare Pages上面。所以成本很低，也很划算！！  

如果你也想要用此模板建立你的博客，可以访问 [原仓库](https://github.com/satnaing/astro-paper)!  

想要快速的开启和部署这个博客，其实需要一点时间，因为他不像wordpress是前后端都在一起。所以需要你仔细的配置每一个文件和参数。
